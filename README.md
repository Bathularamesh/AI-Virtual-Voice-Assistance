🎙️ AI Virtual Voice Assistant


An AI-powered Virtual Voice Assistant built using Python that can understand voice commands and perform various tasks such as searching the web, opening applications, answering questions, and more. This project demonstrates the integration of speech recognition, text-to-speech, and automation to create a smart assistant similar to Siri or Alexa.

🚀 Features

🎤 Voice command recognition

🔊 Text-to-speech response

🌐 Search information on the internet

📂 Open system applications

🕒 Tell current date and time

🔎 Perform Google searches

📧 Send emails (optional)

🎵 Play music

📢 Respond to user queries
